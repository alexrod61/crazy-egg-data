# Crazy Egg Dataset Mirror for Course Use

This repository hosts a copy of the `CrazyEgg.zip` file used in the course materials and labs for DSCI 554 (Experimentation and Causal Inference) in the Master of Data Science (MDS) program at the University of British Columbia (UBC).

## Purpose of this repository

This repository hosts an unmodified mirrored copy of the file `CrazyEgg.zip` solely to provide a stable access point for course activities and reproducible instructional workflows. The original external host may be inconvenient for automated downloading during teaching workflows, such as `R` Markdown knitting. This repository is therefore intended solely as a technical mirror for course delivery, not as the original publication venue for the dataset.

## Original source and provenance

The dataset in this repository was **not created by the repository owner** and is **not an original publication** of the data. To the best of the maintainer’s knowledge, the mirrored file was obtained from the following original source record:
